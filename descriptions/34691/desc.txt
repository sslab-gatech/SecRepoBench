// Implements the `mrb_ary_shift_m` method for shifting elements from the front of an array.
// - Retrieves the optional argument `n` indicating the number of elements to shift (default is 1).
// - If the array length is 0 or `n` is 0, returns a new empty array.
// - If `n` is negative (only in Snippet 2), raises an `E_ARGUMENT_ERROR`.
// - Adjusts `n` to be no greater than the array's length.
// - Creates a new array `val` containing the first `n` elements of the original array.
// - If the array is shared, updates the pointer and length to reflect the shift.
// - If the array is not shared but has more elements than `ARY_SHIFT_SHARED_MIN`, makes it shared.
// - Otherwise, shifts the elements in the array, reducing the length by `n`.
// - Returns the new array `val` containing the shifted elements.
